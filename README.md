# Browser Extensions Manager

A responsive web application for managing browser extensions. This project focuses on dynamic DOM manipulation, CSS variables for theming, and semantic, accessible HTML.

## 🚀 Key Features

* **Dynamic Content:** Extension lists are rendered dynamically from a local JSON file.
* **Theme Switching:** Seamless transition between light and dark modes using CSS variables.
* **Responsive Design:** A mobile-first layout that adapts gracefully to tablets and desktops using CSS Grid and Flexbox.
* **Accessibility (A11y):** Optimized for assistive technologies with ARIA labels, semantic HTML tags, and intuitive keyboard navigation.
* **Filtering:** Functional filtering system to sort extensions by status (All, Active, Inactive).

## 🛠 Technologies

* **HTML5:** Semantic markup.
* **CSS3:** CSS Variables, Grid, Flexbox, Media Queries.
* **JavaScript (ES6+):** DOM manipulation, Event handling, and asynchronous data fetching.
